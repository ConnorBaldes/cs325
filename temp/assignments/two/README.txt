Language: C++

1. mergesort3.cpp:
    -Compile: g++ -std=c++11 mergesort3.cpp -o mergesort3
    -Run: ./mergesort3

    *Note: data.txt must exist in working directory

3. mergesort3Time.cpp:
    -Compile: g++ -std=c++11 mergesort3Time.cpp -o mergesort3Time
    -Run: ./mergesort3Time n(size of array)

    *Note: When running executable an integer must be provided on the command line
           this will be the "n" value aka the size of the array to be created.
