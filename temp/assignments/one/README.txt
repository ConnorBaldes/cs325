This is the read me explaining how to execute program 1 written by Connor Baldes:

1. insertsort.cpp:
    -Compile: g++ -std=c++11 insertsort.cpp -o insertsort
    -Run: ./insertsort

    *Note: data.txt must exist in working directory

2. mergesort.cpp:
    -Compile: g++ -std=c++11 mergesort.cpp -o mergesort
    -Run: ./mergesort

    *Note: data.txt must exist in working directory

3. insertTime.cpp:
    -Compile: g++ -std=c++11 insertTime.cpp -o insertTime
    -Run: ./insertTime n(size of array)

    *Note: When running executable an integer must be provided on the command line
           this will be the "n" value aka the size of the array to be created. 

3. mergeTime.cpp:
    -Compile: g++ -std=c++11 mergeTime.cpp -o mergeTime
    -Run: ./mergeTime n(size of array)

    *Note: When running executable an integer must be provided on the command line
           this will be the "n" value aka the size of the array to be created.
